# Add Contributor Profile Dashboard

Labels: `frontend`, `good first issue`

## Summary

Create a profile view for contributors that shows reserved, submitted, released, and refunded bounties along with total earnings.

## Why It Matters

- Gives contributors a reason to return to the product
- Makes the ecosystem feel more alive
- Offers a clean first issue with visible UI impact

## Acceptance Criteria

- Contributors can filter by status
- Released payout totals are grouped by asset
- Empty and loading states are polished
- The page works on mobile and desktop
